# CueGuide — AI-Driven Daily Routine Assistant for Early Dementia

## Feature Statement

**Enable people with early-stage dementia to complete daily routines independently through AI-generated, personalized step-by-step prompts — reducing caregiver intervention by an estimated 40-60% for routine activities while preserving patient autonomy and dignity.**

---

## Epic: Personalized Routine Prompting Engine

### Feature 1: Routine Creation & Step Decomposition

**As a** caregiver,
**I want to** create daily routines broken into simple, sequential steps with optional media (photos, audio, icons),
**So that** my loved one receives clear guidance tailored to their specific habits and environment.

**Acceptance Criteria:**
- Caregiver can create a routine with a name, category (hygiene, meals, medication, exercise, social), and scheduled time
- Each routine contains 2-15 ordered steps
- Each step has: instruction text (max 50 words), optional photo/icon, optional audio recording of caregiver's voice
- Caregiver can reorder, edit, or remove steps after creation
- System suggests common routine templates (Morning Hygiene, Medication, Meal Prep, Bedtime) as starting points
- Routines can be scheduled as daily, specific days of week, or one-time
- Maximum 12 active routines per day (cognitive load limit based on clinical guidance)

**Dependencies:** User authentication, cloud storage for media assets, routine data model
**Story Points:** 8

---

### Feature 2: AI-Powered Adaptive Prompting

**As a** person with early dementia,
**I want to** receive step-by-step prompts at the right time, in a calm and familiar format,
**So that** I can complete my daily activities without needing someone to remind me.

**Acceptance Criteria:**
- When a routine is due, the app transitions to full-screen "Focus Mode" showing one step at a time
- Each prompt displays: large text (minimum 24pt), optional image, audio playback button, and a single "Done" button
- If no interaction within configurable timeout (default 2 minutes), the system replays the prompt with audio
- After 3 unanswered prompts, the system sends a caregiver alert (not patient-facing — avoids frustration)
- AI personalizes prompt language based on patient profile (preferred name, tone, familiar references)
- AI adjusts prompt timing based on historical completion patterns (e.g., if patient consistently starts morning routine at 8:15 instead of 8:00, system learns and shifts)
- Completion time per step is logged for adaptive learning
- Patient can tap "Help" to hear an expanded explanation generated by AI
- Patient can tap "Skip" to move to next step (logged for caregiver review)

**Dependencies:** LLM API integration (Claude/GPT), notification service, routine scheduling engine
**Story Points:** 13

---

### Feature 3: Caregiver Dashboard & Monitoring

**As a** caregiver,
**I want to** see how my loved one is progressing through their routines, get alerts when they need help, and review patterns over time,
**So that** I can provide support only when truly needed and track cognitive trends.

**Acceptance Criteria:**
- Dashboard shows today's routines with status: upcoming / in progress / completed / missed / partially completed
- Real-time view: when patient is in a routine, caregiver sees which step they're on and time elapsed
- Alerts sent via push notification when: routine missed (configurable grace period), patient stuck on a step > 5 minutes, patient taps "Help" button
- Weekly summary report showing: completion rates per routine, average time per routine (trend over weeks), most-skipped steps, time-of-day patterns
- Caregiver can send an encouragement message that appears on patient's screen ("You're doing great, Dad!")
- Alert fatigue prevention: caregiver sets quiet hours and maximum alerts per hour

**Dependencies:** WebSocket/real-time sync, push notification service, analytics data pipeline
**Story Points:** 8

---

### Feature 4: Contextual AI Cue Generation

**As a** person with early dementia,
**I want** the prompts to feel natural and connected to my life,
**So that** I feel oriented and calm rather than confused by generic instructions.

**Acceptance Criteria:**
- AI generates contextual cues using patient profile data: name, preferred nicknames, familiar objects, room descriptions, pet names, family photos
- Morning prompts include: current day, date, weather, and any upcoming events ("Good morning, Robert. It's Tuesday, May 6th. Sunny today. Your daughter Sarah is visiting at 3 PM.")
- Medication prompts include: pill description, purpose in plain language, and visual reference ("Time for your blue heart pill — it helps keep your blood pressure healthy")
- AI regenerates prompt language periodically to avoid habituation (same instruction, varied phrasing)
- All AI-generated content is reviewable by caregiver before going live (approval queue)
- Fallback to static prompts if AI service is unavailable (offline mode)

**Dependencies:** LLM API, patient profile data model, weather API, calendar integration
**Story Points:** 8

## Business Value

### Direct Value
- **Caregiver time savings:** Early dementia caregivers spend 5+ hours/day on supervision (CDC). If CueGuide handles 40% of routine reminding, that's 2 hours/day returned — worth $50-75/day at home care aide rates.
- **Delayed institutional care:** Maintaining independence delays nursing home placement by 6-18 months (estimated $60K-180K savings per family in the US).
- **Caregiver mental health:** Reducing constant vigilance reduces burnout — the #1 reason caregivers abandon home care.

### Market Context
- 6.9 million Americans living with Alzheimer's (Alzheimer's Association, 2024)
- Global dementia care market: $12.2B (2023), projected $22.8B by 2030
- Existing apps (MemoryMinder, Remindee, CareZone) focus on simple reminders — none use AI for adaptive, personalized step-by-step prompting
- AdaptAILabs is uniquely positioned: AI-first company with healthcare vertical expertise

### Revenue Model Alignment
- B2C: $14.99/month family subscription (patient + up to 3 caregivers)
- B2B: Care facility license ($99/month per 10 patients)
- B2B2C: Insurance/health system partnership (per-patient-per-month model)

## Proposed Technology Stack

### Frontend
- **React Native** (Expo) — cross-platform iOS/Android from single codebase
- **Large-print design system** — minimum 24pt body text, high contrast (WCAG AAA), single-action screens
- **Offline-first architecture** — routines and prompts cached locally, syncs when connected

### Backend
- **Supabase** (PostgreSQL + Auth + Realtime) — HIPAA-eligible with BAA
- **Edge Functions** (Deno) — lightweight API endpoints for AI orchestration
- **Row-Level Security** — patient data isolated per caregiver account

### AI Layer
- **Claude API (Anthropic)** — prompt generation, contextual cues, adaptive language
- **Prompt engineering pipeline:** Patient profile → routine context → step template → personalized output
- **Heuristic scheduling engine** — weighted moving average of completion times, adjusts ±15 min/week max
- **Future:** Fine-tuned model on dementia communication patterns (Phase 2)

### Infrastructure
- **Supabase Cloud** (SOC 2 Type II, HIPAA BAA available)
- **Firebase Cloud Messaging** — push notifications
- **ElevenLabs API** — natural voice TTS for prompts (or device-native fallback)
- **Sentry** — error monitoring
- **PostHog** — analytics (self-hosted for HIPAA)

## Risks & Mitigations

| Risk | Likelihood | Impact | Mitigation |
|---|---|---|---|
| Patient ignores/fears the device | High | High | Gradual onboarding. Caregiver introduces as "your helper." Familiar voice recordings. No alarms — gentle chimes only. |
| Cognitive decline outpaces app utility | Medium | High | Graceful degradation: app transitions to caregiver-only mode. Simplified interface per stage. |
| AI generates confusing/inappropriate prompts | Low | Critical | All AI output passes through safety filter. Caregiver approval queue. Fallback to static prompts. |
| HIPAA compliance gaps | Low | Critical | Supabase BAA. Encryption at rest + transit. No PHI in logs. Annual compliance audit. |
| Caregiver setup burden | High | Medium | Pre-built routine templates. 5-minute onboarding wizard. AI suggests routines from initial questionnaire. |
| Internet connectivity issues | Medium | Medium | Offline-first: all routines + prompts cached. Syncs when reconnected. |
